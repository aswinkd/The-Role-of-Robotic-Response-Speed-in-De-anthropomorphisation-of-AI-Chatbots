# The-Role-of-Robotic-Response-Speed-in-De-anthropomorphisation-of-AI-Chatbots
IEEE ISTAS paper exploring how chatbot response timing (instant vs. delayed)  affects user anthropomorphism. Compares a Standard Chatbot (1.5s fixed) vs.  Dynamic Chatbot (0.5s factual / 3s emotional) using RoBERTa, Flask, React.js,  and Firebase. Published by Heriot-Watt University.
